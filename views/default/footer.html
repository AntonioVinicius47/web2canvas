Recursos

Design Responsivo

O Web2Canvas é criado para fornecer uma melhor visualização e experiência, além de fácil leitura e navegação. Acesse de um dispositivo móvel, tablet ou celular, verá que este site se adequa, o conteúdo se reorganiza de acordo com tamanho do navegador.

Open Source

O Web2Canvas é feito com o Web2py, um framework de Código Aberto para desenvolvimento de aplicações web rápidas, escaláveis, seguras e portáteis.

Cross-platform

O Web2Canvas é uma aplicação Web que roda em vários navegadores Web ao mesmo tempo que pode ser acessível por diversos Sistemas Operacionais como Windows, Linux, Mac e dispositivos móveis como Android e IOS.
